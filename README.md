# Flappy Bat 🦇

A Flappy Bat game built in C++ using Raylib.

## Features
- Pixel-art animated bat with 3 wing states
- Particle effects on flap and death
- Procedurally generated sound effects (no audio files needed)
- Twinkling star background
- Pause menu
- High score system (saved to file)
- Smooth collision detection

## Controls
| Key | Action |
|-----|--------|
| Space / Click | Jump |
| P | Pause |
| M | Toggle sound |
| R / Enter | Restart |

## Tech Stack
- C++
- Raylib

## Setup
1. Install Raylib
2. Clone the repo
3. Compile: `g++ flappybat.cpp -o flappybat -lraylib`
4. Run: `./flappybat`
